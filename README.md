# homecms
租房项目
使用laravel和yii（tp，ci）协同完成开发一个项目

项目描述

公司想要开发一个租房的app项目
要求所有操作必须通过接口实现   使用laravel 写接口，使用yii模拟前端浏览器调用接口

ci 模拟网站前台
